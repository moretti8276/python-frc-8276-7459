# Python para FRC

Este repositório contém uma documentação sobre programação de robôs FRC em Python, com foco em **RobotPy**.

O material explica, como funcionam alguns dos principais conceitos usados no código de um robô, incluindo:
- Estrutura básica de um projeto RobotPy
- Sistema de tração tanque
- A classe `TimedRobot`
- Programação de mecanismos simples
- Introdução ao Command-Based
- Uso de NetworkTables

> Documentação desenvolvida pelas equipes **Optron 8276** e **TTaubatexas Robotics 7459**
